# Imágenes de Plataforma - Landing Page

Esta carpeta contiene las capturas de pantalla o mockups de TrackMySign para la sección "La plataforma para tu negocio".

## Imágenes necesarias:

1. **dashboard.webp** - Captura del panel principal/dashboard
   - Mostrar métricas, gráficos, resumen de actividad
   - Tamaño recomendado: 1200x800px o 16:9

2. **cotizaciones.webp** - Captura del módulo de cotizaciones
   - Mostrar lista de cotizaciones o formulario de creación
   - Tamaño recomendado: 1200x800px o 16:9

3. **seguimiento.webp** - Captura del seguimiento de órdenes
   - Mostrar timeline, estados, o kanban de órdenes
   - Tamaño recomendado: 1200x800px o 16:9

## Formato recomendado:
- **WebP** para mejor compresión y calidad
- Resolución mínima: 1200px de ancho
- Peso máximo por imagen: ~200KB
